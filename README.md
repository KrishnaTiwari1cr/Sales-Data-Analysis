# 📊 Business Performance Analysis

A comprehensive sales and financial analysis project built using Excel. This project demonstrates end-to-end data analysis including data cleaning, transformation, dashboarding, and actionable business insights.

---

## 📌 Sales Analysis

### 🎯 Project Objectives
- Analyze customer trends using [Customer Performance Report](https://github.com/KrishnaTiwari1cr/Sales-Data-Analysis/blob/main/Customer%20Performance%20Report.pdf)
- Compare targets vs actuals via [Market Performance vs Target](https://github.com/KrishnaTiwari1cr/Sales-Data-Analysis/blob/main/Market%20Performance%20vs%20Target.pdf)
- Identify top-performing products using [Top 10 Products](https://github.com/KrishnaTiwari1cr/Sales-Data-Analysis/blob/main/Top%2010%20Products.pdf)

### 📈 Key Insights
- Identified revenue-driving customers and markets  
- Highlighted underperforming regions vs targets  
- Discovered top-performing products contributing to maximum revenue  

### 💡 Business Value
- Improves sales strategy and targeting  
- Enables data-driven decision-making  
- Enhances customer segmentation
